# BATTLE-OF-ATMOSPHERE

## Template

|Gitmoji|入力|使用時|
|:-:|:-:|:-:|
|:art:|```:art:```|コードの構造の改善|
|:fire:|```:fire:```|コードやファイルの削除|
|:bug:|```:bug:```|バグの修正|
|:sparkles:|```:sparkles:```|新機能の追加|
|:tada:|```:tada:```|初期コミット|
|:construction:|```:construction:```|進行中の作業|
|:wrench:|```：wrench：```|設定ファイルの変更|
|:poop:|```:poop:```|改善する必要のある悪いコードを書く
|:rewind:|```:rewind:```|変更を元に戻す|
|:truck:|```:truck:```|ファイルの移動や名前の変更|
|:pencil2:|```:pencil2:```|タイプミスの修正|
|:pencil:|```:pencil:```|コメントの追加|
|:ok_hand:|```:ok_hand:```|コードレビューによるコードの更新|
|:boom:|```:boom:```|大きな変化をもたらす|
|:see_no_evil:|```:see_no_evil:```|.gitignoreを追加|
|:camera_flash:|```:camera_flash:```|画像追加|

## コーディング規約

|||
|:-:|:-:|
|すべてにおいて|波かっこ改行、揃える|
|関数|波かっこ改行|
|自作関数|作業内容、Doxygenコメントを必ずコメント|
|変数名|名詞で小文字のスネークケース|
|関数名|動詞でアッパーキャメル|
|ポインタ|p_変数名|
|構造体名　クラス名　enum名|大文字|
|enumのメンバ変数|大文字のスネークケース|
|構造体　クラス　のメンバ変数|m_変数名|
|型名|すべて大文字|
|#define|大文字のスネークケース|
|インスタンス|型名の小文字|
|FLOAT|整数であっても小数で表し、fを付ける|
|テクスチャー名|小文字のスネークケース|
|for|イテレーターはi、j、kの順に使用する|
|const char*|_T必須|
